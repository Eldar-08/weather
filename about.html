About this app

This is a small demo weather UI inspired by the provided design mockups. It shows a home/start view, a main weather screen and additional info cards. The pages are static HTML/CSS and can be extended to fetch live data from any weather API.

What you can do

  • Open the weather view to see current temperature and hourly forecast (static).
  • Use the layout as a starting point for integration with a weather API like OpenWeatherMap.
  • Customize colors, icons, and typography in styles.css.
