How it works

  1. Type a location into the search box ‚Äî suggestions appear as you type.
  2. Select a location to fetch coordinates from the Geocoding API.
  3. The forecast API returns current and hourly weather which is shown in the UI.

Data sources

This demo uses the Open-Meteo forecast API and its Geocoding service. You can replace these with any other API (OpenWeatherMap, Meteostat, etc.).
